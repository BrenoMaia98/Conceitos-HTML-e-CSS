Exemplo 01 Moldando elementos

A caixa foi criada utilizando as seguintes propriedades CSS:

height: 100px; altura do elemento
width: 200px; largura do elemento
border-radius: 20px; bordas arredondadas
background-color: #ff0; cor do fundo do elemento

border-color: #f00; Também podemos utilizar nomes como 'red', 'yellow', etc.
border-width: 3px;
border-style: solid;
os três ultimos estilos, de maneira minimizada:
border: 3px solid red
